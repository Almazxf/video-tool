# RenPy Video Tool

Программа для пакетной обработки видео под RenPy: ресайз/обрезка/letterbox, конвертация в
WebM (VP9+Opus, рекомендуется RenPy) / WebM (VP8+Vorbis) / OGV (Theora+Vorbis), обрезка по
времени (вырезание фрагмента), отзеркаливание, прогресс-бар. Portable — без установки,
без следов. Под капотом — ffmpeg (бандл рядом с .exe), сама программа просто рисует окно
и дёргает ffmpeg.

## Шаг 1. Создать репозиторий

1. https://github.com/new
2. Имя, например `renpy-video-tool`, Public, ничего не добавлять при создании

## Шаг 2. Залить обычные файлы (через браузер)

На главной странице нового репозитория — "uploading an existing file" (или Add file → Upload files).
Перетащи туда из распакованного архива: `go.mod`, `main.go`, `ffmpeg_runner.go`, `README.md`, `.gitignore`
(сами файлы, не папку). Внизу — Commit changes.

## Шаг 3. Добавить workflow-файл (важно: НЕ через перетаскивание папки)

Папка `.github` при перетаскивании часто "теряется" в браузере — поэтому создаём вручную:

1. На главной странице репозитория → **Add file → Create new file**
2. В поле имени файла впиши целиком: `.github/workflows/build.yml`
   (печатай по одному символу — после каждого `/` GitHub сам формирует папки)
3. Вставь содержимое файла `build.yml` из архива (открой его в блокноте, скопируй всё)
4. Commit changes

## Шаг 4. Дождаться сборки

1. Вкладка **Actions** — сборка идёт дольше, чем у фото-инструмента (3-7 минут), потому что
   скачивается ffmpeg (~80-100 МБ)
2. Если упадёт с ошибкой публикации релиза (как было в фото-проекте) — это лечится тем же
   способом: Settings → Actions → General → Workflow permissions → **Read and write permissions** → Save,
   потом Actions → build.yml → Run workflow
3. ✅ зелёный → **Releases** → скачивай `RenPyVideoTool-windows.zip`
4. Внутри zip — три файла: `RenPyVideoTool.exe`, `ffmpeg.exe`, `ffprobe.exe`.
   **Важно:** все три должны лежать в одной папке на флешке — программа ищет ffmpeg рядом с собой.

## Использование

1. Распакуй все три файла из zip в одну папку на флешке
2. Запусти `RenPyVideoTool.exe`
3. Перетащи видео или папку с видео в окно (или кнопка "Выбрать видео или папку...")
4. Настрой разрешение / обрезку кадра / формат / качество / отзеркаливание / обрезку по времени
5. "Начать обработку" — прогресс-бар покажет и текущий файл, и % внутри него
6. Результат — в подпапке `output` рядом с .exe
